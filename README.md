# CareConnect - Healthcare Support Platform for NGOs

CareConnect is an AI-powered healthcare support platform designed specifically for NGOs and healthcare organizations to streamline patient support requests and volunteer coordination.
The platform connects patients in need with healthcare professionals and volunteers, providing immediate AI-driven insights and facilitating coordinated care.

##  Tech Stack

### Frontend
- **React** - JavaScript library for building user interfaces
- **Vite** - Fast build tool and development server
- **Tailwind CSS** - Utility-first CSS framework for rapid UI development
- **React Router DOM** - Declarative routing for React applications
- **Axios** - Promise-based HTTP client for API requests

### Backend
- **Node.js** - JavaScript runtime environment
- **Express.js** - Web application framework for Node.js
- **MongoDB** - NoSQL database for storing patient and volunteer data
- **Mongoose** - Object Data Modeling (ODM) library for MongoDB
- **CORS** - Cross-Origin Resource Sharing middleware
- **Dotenv** - Environment variable management

### AI Features
- **Rule-Based AI System** - Intelligent analysis of healthcare concerns
- **Priority Classification** - Automatic categorization of requests by urgency
- **Healthcare Insights** - Context-aware guidance and recommendations

CareConnect leverages artificial intelligence to provide immediate, intelligent responses to healthcare concerns:

### AI Analysis Capabilities
- **Urgency Detection**: Identifies emergency vs. non-emergency situations
- **Category Classification**: Groups concerns into medical help, medicine support, and mental health
- **Priority Scoring**: Assigns priority levels (High, Medium, Low) based on concern severity
- **Guidance Provision**: Offers appropriate next steps and resources
- **Natural Language Processing**: Understands various expressions of healthcare needs

### AI Response Logic
The AI system uses a comprehensive rule-based approach:
- **Medical Emergencies**: Detects urgent symptoms and recommends immediate care
- **Medicine Requests**: Provides guidance on prescription assistance and pharmacy resources
- **Mental Health Concerns**: Offers mental health resources and support pathways
- **General Healthcare**: Provides information on healthcare services and support options
